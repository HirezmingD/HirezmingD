### 🔬 研究方向

**AI + 国土资源管理、空间规划与公共治理**

* **资源评估与资产核算** — 房地产评估、土地评估、矿业权评估、资产评估
* **国土空间规划** — 三区三线划定、耕地保护、生态保护红线、土地整治、乡村振兴规划、公园城市建设
* **生态环境与咨询服务** — 生态修复、环境咨询、土地整治、生态保护
* **GeoAI 与空间建模** — 可解释机器学习、InVEST、PLUS、空间计量在生态系统服务与碳动态中的应用
